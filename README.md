# Hexo Theme Shoka

## Usage

1. Clone this repository

``` bash
# cd your-blog
git clone https://github.com/amehime/hexo-theme-shoka.git ./themes/shoka
```

2. update theme fragment as `shoka`.
3. Install plugin
  - hexo-renderer-multi-markdown-it
  - hexo-autoprefixer
  - hexo-symbols-count-time
  - hexo-feed
